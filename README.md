# AI Study Assistant

## Setup
1. Install Python dependencies: `pip install -r requirements.txt`
2. Install React dependencies: `npm install`
3. Run backend: `python backend.py`
4. Run React app: `npm start`
5. Enter topic and generate study materials

## Files
- `backend.py` - Flask server with Gemini AI
- `src/App.js` - React frontend
- `requirements.txt` - Python dependencies
- `package.json` - React dependencies
